    SSNO and PROV-O Together At Last

    Project maintained by dcorsar Hosted on GitHub Pages — Theme by mattgraham

    Welcome to GitHub Pages.

    This is the page for the SSN-PROV ontology, an alignment betweent the W3C Semantic Sensor Network (SSN) ontology SSNO and the W3C Provenance Working Group PROV-O ontogies. The alignment extends the expressive capability of the SSNO for recording observations and system deployments, enabling more comprehensive historical information to be described than is possible using SSNO alone.

    For further details about the design of the ontology, please see

    Compton, M. and Corsar, D. and Taylor, K. Sensor Data Provenance:SSNO and PROV-O Together At Last. Joint Proceedings of the 6th International Workshop on the Foundations, Technologies and Applications of the Geospatial Web and 7th International Workshop on Semantic Sensor Networks. Vol 1404 CEUR Workshop Proceedings. 2014.
